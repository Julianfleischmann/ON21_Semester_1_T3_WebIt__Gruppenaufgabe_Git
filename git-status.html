GIT-Status

Allgemeine Beschreibung zu diesem Befehl

git status ist ein Befehl, welcher den aktuellen Status auf dem aktuellen Branch auf dem man sich befindet anzeigt. Der Befehl dient zu übersicht um sich anzeigen zu lassen. Was hat man bis jetzt geändert. Ebenso bekommt man seinen aktuellen Branch angezeigt.

GIT-status

Ein paar optionen für git status:

--short : Ausgabe in Kurz Format.
--long : Standard. Zeigt das Lange Format an.
--untracked-files=no : Zeigt keine untracked Files an.
--untracked-files=all : Zeigt alle untracked Files als auch die individuellen Dateien an.
