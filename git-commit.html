GIT-Commit

Allgemeine Beschreibung zu diesem Befehl

git commit ist ein Befehl, welcher die aktuellen Änderungen auf den Branch comittet auf dem man sich gerade befindet. Der Commit Befehl ist ein zentraler Befehl von Git. Dieser übernimmt die Änderung auf dem lokalen Branch. Später kann dann der Commit auf den Remote-Server gepushed werden.

GIT-Commit best practice!

Bei git commit gilt:

Viele kleine Commits sind besser als wenig große Commits.
Ebenso sollte der Commit eine Aussagekräftiges Nachricht haben. Diese soll den Commit kurz und Präzise beschreiben..
Fokusiere dich bei dem Commit auf eine Aufgabe. Für verschiedene Probleme verschiedene Commits!.
